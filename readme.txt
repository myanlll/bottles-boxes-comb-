>sise_komb_kutu.py dosyasını çalıştırmak için dilerseniz vscode indirip sağ üstte bulunan ufak oynatma tuşuyla
başlatıp gereken girdileri klavye üstünden girebilirsiniz.

>sise_komb_kutu.py dosyasını çalıştırdığınızda sizden sırasıyla şişe sayısı ve kutu kapasitesi girdilerini isteyecektir.


>Girdileri dilediğinizce (şişe çeşit sayısını çok arttırmayın bence :) ) girdikten sonra, betik size önce kaç kombinasyon bulduğunu ardından bu kombinasyonları ve 
ardından okuma kolaylığı olması açısından tekrardan kaç kombinasyon bulduğunu yazacaktır.

>Kodun çalışma mantığı olası tüm kombinaysonları bulup ardından kurallara uymama durumuna göre ilerlediğinden ötürü
verimli çalışmamaktadır. Bruteforce üstünden kuralları uygulayarak filtreleme sisteminden geçtiğinden dolayı sonucu, olması gerekenden biraz daha geç bulmaktadır. 
Ama önünde sonunda size doğru sonucu verecektir.

> To run the `bottle_comb_box.py` file, you can download vscode and use the small play button in the top right to start and enter the required inputs via the keyboard.

> When you run the `bottle_comb_box.py` file, it will first ask you for the number of bottles and then the box capacity.

> After entering the inputs as desired (I suggest not increasing the number of bottle types too much :) ), the script will first tell you 
how many combinations it found, then list these combinations, and for ease of reading, it will again tell you the total number of combinations found.

> The code's working principle is not efficient because it finds all possible combinations first and then checks if they adhere to the rules. It uses brute force to apply the rules and filter 
the combinations, so it takes a bit longer than necessary to find the result. However, it will eventually give you the correct result.
